Ihre Funktion sollte eine mehrzeilige Zeichenkette mit eindeutigen Kleinbuchstaben in alphabetischer Reihenfolge zurückgeben. Die Buchstaben sollten in Form eines hohlen Rhombus (Raute) entlang der Kanten, beginnend mit dem oberen Scheitelpunkt, angeordnet werden.

Zu diesem Zweck werden Ihnen drei Werte vorgegeben (für deren Einschränkungen siehe Vorbedingungen):

  • side - anzahl der Buchstaben auf jeder Seite des Rhombus;
  • length - gesamtzahl der Buchstaben, die Sie verwenden sollen;
  • cw - uhrzeigersinn-Flagge, die angibt, in welche Richtung die Buchstaben gehen müssen (wenn {% if is_js %}t{% else %}T{% endif %}rue - buchstaben im Uhrzeigersinn platziert werden sollen).
