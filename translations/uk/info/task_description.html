Твоя функція повинна повернути багатолінійний рядок з унікальних малих літер в алфавітному порядку. Символи повинні бути розміщені у вигляді порожнистого ромба (діаманту), вздовж його країв, починаючи з верхньої вершини.

Для цього тобі дано три значення (їх обмеження дивись у Передумовах):

  • side - кількість букв на кожній стороні ромба;
  • length - загальна кількість букв, які слід використати;
  • cw - прапорець руху за годинниковою стрілкою, що показує в якому напрямку повинні йти літери (якщо {% if is_js %}t{% else %}T{% endif %}rue - букви повинні бути розміщені в напрямку за годинниковою стрілкою).

Якщо не вистачає length для заповнення форми, доповни її з допомогою "*". Ромб не повинен мати пропуски праворуч. Ось приклад з аргументами та очікуваним результатом:

Вхідні дані: Три аргументи: сторона як ціле число ({% if is_js %}number{% else %}int{% endif %}), кількість як ціле число ({% if is_js %}number{% else %}int{% endif %}) і прапорець руху за год. стрілкою як логічне значення (bool{% if is_js %}ean{% endif %}).

Вихід: Багатолінійний рядок (str{% if is_js %}ing{% endif %}).
