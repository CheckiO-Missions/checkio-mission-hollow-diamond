Ваша функція повинна повернути багаторядковий рядок унікальних малих літер в алфавітному порядку. Символи повинні бути розміщені у вигляді порожнистого ромба (ромба), по його ребрах, починаючи з верхньої вершини.

Для цього вам задано три значення (обмеження на них див. у розділі Умовах):
