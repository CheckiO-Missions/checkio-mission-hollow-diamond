您的函数应按字母顺序返回由唯一的小写字母组成的多行字符串。这些字符应从顶点开始，沿其边缘排列成空心菱形（菱形）。

为此，您将得到三个值（其约束条件请参见 "先决条件"）：

  • side - 菱形每边的字母数；
  • length - 您应该使用的字母总数；
  • cw - 顺时针方向标志，表示字母必须按什么方向排列（如果 {% if is_js %}t{% else %}T{% endif %}rue - 字母应按顺时针方向放置）。

如果length 不足以完成表格，请用"*" 补充。菱形右侧不应留白。下面是一个包含参数和预期结果的示例：

输入：三个参数：边长为整数 ({% if is_js %}number{% else %}int{% endif %}), 长度为整数 ({% if is_js %}number{% else %}int{% endif %}), 顺时针方向标志为逻辑 (bool{% if is_js %}ean{% endif %}).

输出：多行字符串 (str{% if is_js %}ing{% endif %}).

前提条件：

  • 26 >= length >= 0；
  • side > 1；
  • (side-1)*4 >= length.
