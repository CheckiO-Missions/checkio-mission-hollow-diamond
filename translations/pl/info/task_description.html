Funkcja powinna zwrócić wielowierszowy ciąg unikalnych małych liter w porządku alfabetycznym. Znaki powinny być umieszczone w formie wydrążonego rombu (diamentu), wzdłuż jego krawędzi, zaczynając od górnego wierzchołka.

W tym celu proszę podać trzy wartości (ich ograniczenia znajdują się w sekcji Warunki wstępne):

  • side - liczba liter w każdym boku rombu;
  • length - całkowita liczba liter, których należy użyć;
  • cw - flaga zgodna z ruchem wskazówek zegara, która pokazuje, w którym kierunku muszą iść litery (jeśli {% if is_js %}t{% else %}T{% endif %}rue - litery powinny być ułożone zgodnie z ruchem wskazówek zegara).

Jeśli nie wystarczy length do uzupełnienia formularza, proszę uzupełnić go o "*". Romb nie powinien mieć białych spacji po prawej stronie. Poniżej znajduje się przykład z argumentami i oczekiwanym wynikiem:

Wejście: Trzy argumenty: bok jako liczba całkowita ({% if is_js %}liczba{% else %}int{% endif %}), długość jako liczba całkowita ({% if is_js %}liczba{% else %}int{% endif %}) i flaga zgodna z ruchem wskazówek zegara jako logika (bool{% if is_js %}ean{% endif %}).

Wyjście: Wielowierszowy ciąg znaków (str{% if is_js %}ing{% endif %}).
